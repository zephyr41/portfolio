Quand une Clé USB siphonne votre vie privée
en 30s chrono

by Nils Jaudon

19 Aout, 2024

cyber dev iot

Imaginez une clé USB, discrète et banale, qui, une fois branchée, siphonne silencieusement la vie privée de l'utilisateur. En quelques secondes, elle récupère, photo, message, chaque souvenir partagé.
Avec elle, les secrets les plus profonds d'une personne sont à portée de main.
Et maintenant, imaginez que cette clé USB, capable de tout cela, je l’ai réellement créée.

Tout a commencé avec un projet de web scraping. Pour faire simple, le web scraping est un algorithme qui permet de récupérer massivement des données sur Internet.

Imaginez que vous cherchez à acheter une nouvelle paire de baskets. Vous visitez plusieurs sites pour comparer les prix, lire les avis des clients, et trouver la meilleure offre. Cela peut prendre du temps et nécessiter beaucoup de clics.

Maintenant, imaginez qu’un outil, sans que vous le sachiez, parcourt automatiquement tous ces sites pour vous. Il récolte en quelques secondes tous les avis, prix, et disponibilités pour ce modèle de baskets. Ensuite, il trie les informations et vous présente les meilleures options disponibles.

C’est là que le web scraping entre en jeu. Il récupère des données en grande quantité sur Internet, que ce soit pour des produits, des avis, ou des tendances. Ce puissant outil travaille en coulisses, souvent sans que vous vous en rendiez compte.

Et, donc où est la vie privé dans tout ça ?

Bien maintenant que j'ai introduit le web scraping, je vais vous présenter mon projet. Pour ce projet en Python, l’objectif de base était simplement de récolter des données à partir d'un site web. Cependant, en tant que passionné de cybersécurité, je voulais explorer comment lier ce projet à la cybersécurité.

J'aime beaucoup discuter des dangers de la vie privée sur Internet. Alors en étant sur Instagram sur mon ordinateur, une idée m'est venue. Mon ordinateur utilise un gestionnaire de mots de passe, souvent intégré au navigateur (dans mon cas, Firefox). Je me suis demandé s'il était possible de récupérer les mots de passe enregistrés dans le navigateur et de les utiliser pour accéder à mes comptes, afin d'extraire des informations sensibles et privées (donc, pour faire du web scraping).

En plus du script, j'avais entendu parler de ce qu'on appelle une "rubber ducky" : une clé USB qui se fait passer pour un clavier. J'ai trouvé ce concept fascinant, et il y a longtemps que je voulais réaliser un projet avec. J'ai donc eu l'idée de créer une fausse clé USB qui, une fois branchée, récupère le gestionnaire de mots de passe, extrait les informations dans un fichier, puis utilise un script Python pour se connecter aux comptes de la victime et télécharger ses données.

Comment je m'y suis pris ?

j'ai d'abord divisé le projet en 2 étapes, comment je récolte mes données ? et comment j'obtient les mots de passes ? Pour faire du webscrapping sur des sites connus, cela n'a pas était une mince affaire, en effet, instagram par exemple, obfusque le code de la page.
class=ldazjdlkzaj (non ce n'est pas le chat qui marche sur le clavier c'est un bout du code la page d'instragram)

Bien, pour réaliser mon web scrapping, j'ai utilisé Selenium & Beautifullsoup
se sont des outils complémentaires pour le web scraping. Beautiful Soup simplifie l'extraction et l'analyse des données HTML, tandis que Selenium automatise les interactions avec les navigateurs pour récupérer des données dynamiques. je vais expliquer pourquoi j'ai surtout utilisés Selenium

enfaite, selenium me permet d'avoir un navigateur qui intéragit avec du code, (je peux ouvrir une page sur google et lui dire d'écrire dans la barre de recherche : Quel temps fais t'il ?) Juste avec du code il est donc beaucoup plus simple de l'utiliser pour naviguer et faire des test quand le code source est obfusqués

Bien, je pourrais essayer de dechiffrer le code, mais cela prendrait une éternités, et je n'ai pas une éternité pour mon projet

si les éléments de la page on beau être imcompréhensible, en l'occurence il ne bouge pas, il y'aura toujours : dajlkajzklaj, au meme endroit. j'ai juste à comprendre à quoi signifie dajkljzdlaj pour dire a selenium, tu clique sur dajkljlj (pour par exemple entrez le mot de passe) j'ai répété l'opération jusqu'a trouver les éléments qui m'interessait, Ou entrez le mot de passe, nom d'utilisateur, se connecter, puis une fois connecté, accèder au message privé, et les télécharger. (étape la plus simple car j'ai juste récupèrer tout le code source de la page et donc les messages privés)
